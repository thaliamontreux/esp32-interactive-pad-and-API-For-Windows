# DisplayPad Server

DisplayPad Server is a Windows-only Python application for managing ESP32 touchscreen display pads over Wi-Fi.

Pads can operate as:

- **Button Pads**: secure macro keypads
- **Task Pads**: active program/task switchers

The Windows API owns device configuration, security, layouts, macros, task lists, icons, PIN policies, and update notifications.  
The ESP32 owns touchscreen display, local Wi-Fi setup, PIN entry, diagnostics, and secure communication.

## Primary Hardware

- ESP32-WROOM-32
- ILI9341 touch display
- 240x320 resolution

## Core Goals

- Secure pairing with one-time pairing codes
- Per-device Control Panel PINs up to 8 digits
- Default PIN: `00000000`
- Multiple ESP32 pads supported
- 6 to 32 buttons per pad
- Dynamic layouts generated by the Windows API
- Button Pad and Task Pad modes
- Windows system tray application
- FastAPI backend
- SQLite database
- WebSocket config-update notifications
- Secure signed button events
- Windows-only macro execution using SendInput

## Quick Start

### Installation

```powershell
# Install dependencies
pip install -e .

# Or with the requirements file
pip install -r requirements.txt
```

### Running the Server

**Development mode (no tray, auto-reload):**
```powershell
.\scripts\run_dev_server.ps1
```

**Production mode (with system tray):**
```powershell
python run_server.py
```

### Using the API

The server runs on `http://localhost:7443` by default.

**Health check:**
```bash
curl http://localhost:7443/health
```

**Start pairing (from tray menu or API):**
```bash
curl -X POST http://localhost:7443/api/v1/pairing/start
```

### Project Structure

```
src/displaypad_server/
  api/          - FastAPI route handlers
  core/         - Config, crypto, layout, security
  db/           - Database initialization and connections
  windows/      - Windows-specific code (SendInput, tray, processes)
  main.py       - FastAPI app factory
  launcher.py   - Combined server + tray launcher
```
